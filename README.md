# Aufwand SHK – App-Download

Offizieller Download der **Aufwand SHK**-App (Android) für die Monteure.

## Installieren

1. Diesen Link auf dem Android-Gerät öffnen (oder QR-Code aus der App scannen):
   **[aufwandsapp.apk herunterladen](https://github.com/onetoner/aufwandsapp-download/releases/download/live/aufwandsapp.apk)**
2. Die heruntergeladene `aufwandsapp.apk` öffnen.
3. Falls gefragt: „Installieren aus unbekannten Quellen" für den Browser/Dateimanager erlauben.
4. Auf **Installieren** tippen – fertig.

## Updates

Die App prüft beim Start automatisch auf Updates und bietet „Jetzt aktualisieren" an.
Manuell geht es in der App unter **Einstellungen → App-Version & Update**.

## Hinweis

Die jeweils aktuelle Version liegt unter dem Release-Tag **`live`**. Ältere
Versionen werden privat archiviert, damit bei Problemen auf eine vorherige
Version zurückgerollt werden kann.
